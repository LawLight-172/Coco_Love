# 💖 Valentine Proposal Website

An interactive, animated Valentine proposal website that allows users to personalize romantic messages, names, and text content to create a beautiful digital surprise for their loved ones.  
This project works completely offline (locally) and can also be hosted online easily.

> Made using ❤️ with Kimi v2 AI

---

## 🌟 Highlights

- 💕 Interactive Valentine proposal flow  
- 💌 Editable love letter message system  
- 🎁 Gift animation screens (Teddy, Letter, Chocolate, Surprise)  
- 🎆 Confetti, hearts & particle effects  
- 📱 Fully responsive layout  
- ⚡ Lightweight & fast (No backend required)  
- 🌍 Works offline & online  
- 🧑‍💻 Easy customization for anyone

## ✏️ How To Customize Content

You can personalize everything easily without coding experience.

---

### 💬 Change Main Proposal Text

Open:

```
index.html
```

Find:

```html
Will you be my Valentine?
```

Replace with:

```html
Will you be my Valentine, XYZ? ❤️
```

---

### 💌 Change Love Letter Message

Open:

```
js/script.js
```

Search for:

```js
const letterMessage
```

Example:

```js
const letterMessage = "Write your romantic message here...";
```

Replace it with your custom message.

---

### 🎀 Change Headings & Titles

In `index.html`, modify text like:

```html
<h1>Hey there, Will you be my Valentine?</h1>
```

To:I Love You Coco Will You be my Always Wiffu?

```html
<h1>Hello XYZ, Will you be my Valentine?</h1>
```

---

### 🧸 Customize Gift Messages

Each gift screen has separate content.

You can edit:

- Teddy message  
- Letter content  
- Chocolate message  
- Surprise text  

Inside:

```
index.html
```

or

```
js/script.js
```

## 🌐 Hosting Online (Free Deployment)

You can make your Valentine website public online.

---

### ✅ GitHub Pages (Recommended)

Steps:

1. Open your GitHub repository  
2. Click **Settings**  
3. Go to **Pages**  
4. Select branch: `main`  
5. Select folder: `/root`  
6. Click **Save**  

Your website will be live at:

```
https://valentine-proposal-website.vercel.app/

```
